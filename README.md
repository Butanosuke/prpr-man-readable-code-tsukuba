# リーダブルコード勉強会 in つくば
## 実行環境
C++  
Apple LLVM version 6.1.0 (clang-602.0.53) (based on LLVM 3.6.0svn)

## 実行方法
```
$ c++ recipe.cpp
$ ./a.out data/recipe-data.txt
```
もしくは
```
$ clang++ recipe.cpp
$ ./a.out data/recipe-data.txt
```
もしくはIDを指定
```
$ c++ recipe.cpp
$ ./a.out data/recipe-data.txt 2
```
